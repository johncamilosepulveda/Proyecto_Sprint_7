# 🚗 Proyecto Sprint 7 – Análisis de anuncios de venta de vehículos
## 📖 Descripción del proyecto
Este proyecto consiste en una aplicación web interactiva que permite explorar un conjunto de datos de anuncios de venta de vehículos mediante visualizaciones gráficas.
La aplicación fue desarrollada en Python utilizando Streamlit y desplegada en la nube con Render, lo que permite acceder a ella desde cualquier navegador sin necesidad de instalación.
🔗 Aplicación en producción:  
https://proyecto-sprint-7-779g.onrender.com  
🔗 Repositorio:  
https://github.com/johncamilosepulveda/Proyecto_Sprint_7  
---
## 🎯 Objetivo
El objetivo principal es analizar datos de vehículos en venta para identificar patrones relevantes, como:
- Distribución del kilometraje (odómetro)
- Relación entre el precio y el kilometraje de los vehículos
---
## ⚙️ Funcionalidades principales
La aplicación cuenta con una interfaz sencilla e interactiva basada en dos botones principales:
### 📊 1. Construir histograma
Al hacer clic en este botón, la aplicación genera un histograma que muestra:
- La distribución del kilometraje (odómetro) de los vehículos
- Frecuencia de vehículos en distintos rangos de uso
🔎 **¿Qué permite analizar?**
- Identificar en qué rangos de kilometraje hay más vehículos
- Detectar concentraciones de autos poco o muy usados
---
### 📈 2. Construir gráfico de dispersión
Este botón genera un gráfico de dispersión que muestra:
- Relación entre el precio del vehículo y su kilometraje
🔎 **¿Qué permite analizar?**
- Cómo afecta el uso (kilómetros) al precio
- Identificar tendencias (por ejemplo: a mayor kilometraje, menor precio)
- Detectar valores atípicos (autos muy caros o muy baratos para su uso)
